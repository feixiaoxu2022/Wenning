### ⏳ demo_presentation.pptx
**案例名称**：商业计划演示
**类型**：PowerPoint演示文稿
**建议内容**：
- 约10-15页
- 主题：AI创业公司商业计划
- 包含：封面、问题与机遇、解决方案、市场分析、商业模式、团队介绍、财务预测等
- 设计风格：现代简约，使用渐变色和数据可视化

## 🔧 如何添加文件

1. 将制作好的视频文件重命名为 `demo_video.mp4`，放到本目录
2. 将制作好的PPT文件重命名为 `demo_presentation.pptx`，放到本目录
3. 刷新页面，点击对应的showcase按钮即可预览

## 💡 提示

- 如果demo文件暂时缺失，点击对应的showcase按钮会显示加载失败，这是正常的
- `demo_dashboard.html` 是完整可用的，可以作为参考示例
- 可以根据需要修改 `/static/data/showcase_cases.json` 来调整showcase案例配置
