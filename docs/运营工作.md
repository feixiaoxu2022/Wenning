## 二、产品运营的主要工作内容

### 1. **用户运营：全生命周期用户**

* 了解用户画像、使用场景、反馈痛点
  * 工作频次：
  * 信息来源：

    * 后台数据用户画像、用户行为
    * 调研、用户访谈
    * 外部平台用户观察
  * 产出：

    * 数据监测、分析的数据搭建需求
    * 分析方法
    * 增长机会：机会洞察、产品功能、运营机制（活动设计、用户激励、裂变等用户体系）

### 2. **内容/活动运营：**站内外发生在推广阶段，用起来与持续用阶段发生在站内；

* 找选题：

  * 接上用户分析的洞察、
  * 固定节假日热点，可穷举
  * 当季热门话题，随时变化
* 策划活动：

  * 定目标；
  * 脑爆方案：活动方案有很多固定框架，需要经验做选择
  * 模拟用户行为检查方案的吸引力与逻辑性，以及对目标强达成设计
  * 传播策略：预算划分、选渠道、谈合作、还需要强经验
* 制作物料：

  * 活动prd：发需求与沟通需求
  * 图文视频内容（推送、公告、教程、案例、showcase）：找素材、找示例、写brief、构思过程、设计or 制作、
* 上线追踪：

  * 衡量效果：曝光、转化、留存
  * 危机预案随时ready ：黑产攻击、舆情危机、礼品库存管理、客服支持用户FAQ

### 3. **数据运营**

* 建立关键指标体系（DAU、留存、转化、流失等）
* 日/周/月度监控、分析、复盘
* 通过数据发现问题与机会（比如用户在哪个环节流失）

### 4. **产品协同**

* 和产品经理一起定义功能上线节奏
* 收集用户反馈并形成产品需求
* 参与A/B测试、版本迭代评估
* 产品文案撰写

### 5. **增长/生态运营**

* 搭增长闭环（渠道 → 激活 → 留存 → 传播）：尤其是商业产品，渠道代理分发与效果归因；留存与传播多与前3个运营结合；
* 拓展合作伙伴、开发者或社区生态：准备合作方案、demo物料、历史案例材料、产品介绍材料；最终与前几个运营动作结合呈现。
